## 主要メッセージ

### バリュープロポジション

> Claude Codeを38個のAIエージェントによる自律型開発チームに変える

### キーメッセージ

1. **5分でセットアップ完了** - 技術的ハードルの低さ
2. **完全無料・オープンソース** - 金銭的ハードルなし
3. **38エージェント、56コマンド** - 豊富な機能
4. **172個のMCPツール** - 拡張性の高さ

### 差別化ポイント

- 日本人開発者による日本語対応
- エンタープライズグレードのセキュリティ
- アクティブなコミュニティサポート
